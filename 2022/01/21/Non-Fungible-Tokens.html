Non Fungible Tokens

Quick description: Working on NFT: minting, viewing and transfering
Project state: pending
Link: https://github.com/Lcbc12/NFT

Why NFT

Since 2021, NFT are becoming more and more omnipresent, almost every blockchain project deals with them. Browsing through them you can find games, tokenomics, art, etc.

With such an emergence, I could not pass by without being interested in it.

This is why I started this project: to train myself on this technology and see the potential to be drawn from it.

Versions

I imposed a batch of features on myself through versions to be able to deliver a finished and functional product each time. So, once a version is finished, I start the features of the next one.

Version 1

Version state: Finished

This version was the start of the project, so I wanted to make the very basis about NFT on a blockchain technology that I know well.

For the back-end part, I will use Ganache with Truffle for solidity development. For the front-end part, I will use ReactJS, connected to MetaMask to sign every transaction.

Here is the diagram of this solution:

The main objective was to make the basics features: mint and view.

Thanks to this solution, anybody can mint a new NFT, store the metadata generated in NFT Storage and saving the metadata URI in the blockchain.

The viewer will then retrieve all the metadata URI from the blockchain, load it and load the image into the site.

To sum up:

  • Mint an image, metadata’s URI is stored in the blockchain, metadata is stored in NFT.Storage;
  • View a NFT by recovering the image by doing the reverse process.

Version 2

Project state: Not started yet

For the second version, I want the site to become a marketplace where anybody can sell/buy any NFT she possesses.

I think that I’ll have to work with token economics, not only account.transfer(value) in solidity.

I will try to make the site connect to other cryptowallet and, hence, other blockchain.

To sum up:

  • Marketplace to sell/buy NFT with other tokens;
  • Connection to other cryptowallets.
